Smart Farm Automation

Automasi produksi telur menggunakan Python untuk menganalisis data, membuat grafik, dan menghasilkan laporan secara otomatis.

🚀 Features

- Data analysis dari file CSV
- Visualisasi grafik otomatis
- Generate report otomatis

🛠 Tech Stack

- Python
- Pandas
- Matplotlib

▶️ Cara Menjalankan

python main.py

📊 Output

- Grafik produksi
- File laporan (.txt)

📁 Struktur Project

smart-farm-automation/
│
├── main.py
├── farm.py
├── config.py
├── utils.py
├── data_produksi.csv

📌 Catatan

Project ini adalah simulasi sistem automasi farm berbasis data.
